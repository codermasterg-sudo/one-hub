## 🎯 按场景查找

### 场景 1：首次部署 One-Hub

1. 阅读 [Docker部署指南](./Docker部署指南.md)
2. 使用 `make install` 一键部署
3. 配置 Clash（参考 [clash/README.md](../clash/README.md)）
4. 配置 Claude 代理（参考 [CLIProxy集成指南](./CLIProxy集成指南.md)）

### 场景 2：配置 Clash 订阅

1. 阅读 [clash/README.md](../clash/README.md)
2. 复制订阅配置：`cp clash/config-subscription.yaml clash/config.yaml`
3. 编辑订阅链接
4. 使用 `make clash-update` 更新订阅

### 场景 3：配置 Claude 多账号代理

1. 阅读 [CLIProxy集成指南](./CLIProxy集成指南.md)
2. 启动服务：`make up`
3. 登录账号：`make cliproxy-login-multi`
4. 在 One-Hub 中配置渠道

### 场景 4：无浏览器环境部署

1. 阅读 [Claude无浏览器登录](./Claude无浏览器登录.md)
2. 选择合适的认证方式（setup-token 或凭证复制）
3. 按指南完成认证

### 场景 5：日常运维

1. 参考 [Makefile使用指南](./Makefile使用指南.md)
2. 使用 `make help` 查看所有命令
3. 使用 `make health` 进行健康检查
4. 使用 `make backup` 定期备份

## 🔍 快速参考

### 常用命令

```bash
# 查看所有命令
make help

# 一键部署
make install

# 启动服务
make up

# 查看状态
make ps

# 查看日志
make logs

# 健康检查
make health
```

### Clash 相关

```bash
# 配置 Clash
make clash-setup

# 更新订阅
make clash-update

# 测试代理
make clash-test

# 打开 Dashboard
make clash-dashboard
```

### CLIProxy 相关

```bash
# 登录多账号
make cliproxy-login-multi

# 查看状态
make cliproxy-status

# 刷新令牌
make cliproxy-refresh

# 查看统计
make cliproxy-stats
```
